Sobre o TBCA

A Tabela Brasileira de Composição de Alimentos (TBCA) é uma base de dados que contém informações sobre a composição nutricional dos alimentos consumidos no Brasil.

Esta plataforma web permite consultar facilmente os dados nutricionais de diversos alimentos, facilitando o acesso a informações importantes para nutricionistas, pesquisadores e o público em geral.

API Disponível

Os dados também estão disponíveis através de uma API REST:

GET /api/alimentos?termo=arroz&categoria=cereais&limite=10

Parâmetros:

  • termo: Termo para busca (opcional)
  • categoria: Categoria de alimentos (opcional)
  • limite: Número máximo de resultados (padrão: 50)
